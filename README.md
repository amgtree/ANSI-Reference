# ANSI Terminal Reference

A comprehensive reference collection for ANSI/VT terminal escape sequences, featuring interactive HTML documentation with formatted sequences, hex bytes, descriptions, and terminal previews. Includes multiple iterations and design variations optimized for quick lookup and copy-paste workflows.

# ASCII Reference

Page showing all the different ascii characters, their decimal, hex, and alt codes.  Different modes 
on the top change how the content is copied to the clipboard.  There is a list view and a table view.
With search and group based filtering.
